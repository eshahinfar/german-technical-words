# Technical German Leitner Trainer (PHP + SQLite) — v3 (All Upgrades)

## What's included
- Topic-by-topic study + Leitner due reviews (Boxes 1–5)
- Quiz modes:
  - Typed answer (German→English / English→German)
  - Multiple-choice (4 options)
- Smart grading with Strict vs Lenient toggle
- New-cards pacing ("New cards/day" setting respected in topic study)
- Learned words list with search + filters
- Dashboard with mastery by topic + box distribution
- Streaks & performance charts (no external libs)
- Backup export/import (progress + settings + review log)
- Login/Register (password-hashed)
- Audio buttons (browser TTS)
- Mobile-friendly UI

## Run locally
php -S localhost:8080
Open http://localhost:8080
